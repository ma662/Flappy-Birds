# Flappy-Birds
MERN Stack Flappy Bird browser game built with React and Passport/bCrypt for authentication security.

![](flappy-birds-demo.gif)

Deployed here: https://flappy-birds-deploy.herokuapp.com

## Technologies:
- HTML5
- CSS
- NodeJS
- ReactJS
- jQuery
- Express
- MongoDB
- Mongoose
- bCrypt
- PassportJS
